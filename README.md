# @vdphan/capacitor-app-update

App update plugin

## Install

To use npm

```bash
npm install @vdphan/capacitor-app-update
````

To use yarn

```bash
yarn add @vdphan/capacitor-app-update
```

Sync native files

```bash
npx cap sync
```
